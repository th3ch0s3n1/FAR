# Farmakologie

**Farmakologie** je odvětví medicíny a biologie zabývající se studiem účinku léčiv. Léčivo můžeme definovat jako uměle vytvořenou, přírodní či endogenní látku,  která biochemickou a/nebo fyziologickou cestou ovlivňuje buňku, tkáň,  orgán či dokonce celý organismus. 

## Obsah

- Obecná farmakologie
  - Farmakokinetika
    - Způsob podání léčiva
    - Absorpce
    - Distribuce
    - Biotransformace
    - Exkrece

  - Farmakodynamika
    - Nespecifické účinky léčiv
    - Specifické účinky léčiv

  - Farmakogenetika a farmakogenomika
  - Farmakoekonomika
  - Vývoj, registrace, druhy a klasifikace léčiv
  - Lékové interakce a nežádoucí účinky léčiv
  - Farmakoterapie a těhotenství
  - Farmakoterapie a stáří

- Speciální farmakologie
